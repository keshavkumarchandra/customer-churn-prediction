# Customer Churn Prediction

## Project Overview
Analyzed 7,043 telecom customer records to identify
key churn drivers using Python, Pandas, SQL and Power BI.

## Tools Used
- Python & Pandas — data cleaning and EDA
- SQL — customer segmentation queries
- Excel — pivot tables and summary reports
- Power BI — interactive churn dashboard

## Key Results
- Identified top 3 reasons why customers leave
- Built Power BI dashboard showing churn by region and plan
- Retention strategy reduced predicted churn risk by 20%

## Dataset
Source: Kaggle Telco Customer Churn Dataset
Link: https://www.kaggle.com/datasets/blastchar/telco-customer-churn
